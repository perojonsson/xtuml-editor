What’s New in BridgePoint 4.1.16

Release Notes

 

Compare and Merge

·         A number of issues with compare and merge were identified and fixed in this release.  Please see the release notes for the detailed list of fixes.  In particular, one of the issues had the potential to cause model corruption.  Hence, users are strongly encouraged to use this patch release in order to avert any model data problems.

 

·         Comparing and merging missing elements is now easier due to an enhancement that adds a placeholder for each missing element.  This allows for easier identification of where a model element will be placed after merging.

 

User Interface

 The feature that allows multiple canvas element tooltips to show at the same time has been enhanced to work in all situations.
